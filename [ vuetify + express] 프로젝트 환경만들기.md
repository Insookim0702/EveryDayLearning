# 1. express 설치
```
npm install express -generator
express --view=ejs backend
npm install
nodemon bin/www
```

# 2. vue 설치
```
npm install @vue/cli -generator
vue create frontend
cd frontend
npm run serve
```

# 3. vuetify 설치
```
vue add vuetify
```
# 4. vue build 경로 설정
```
vue ui
```

# 5. ``` npm run build ```

# 6. localhost:3000
